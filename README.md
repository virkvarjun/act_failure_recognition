# ACT Failure Recognition

Research project for failure recognition in robot policies using the ACT (Action Chunking with Transformers) framework.

## Project Structure

```
failure_policies_research/
├── src/                    # Source code
│   ├── __init__.py
│   ├── models/             # Model architectures
│   ├── data/               # Data loading and processing
│   └── utils/              # Utility functions
├── experiments/            # Experiment scripts and configs
├── notebooks/              # Jupyter notebooks for exploration
├── tests/                  # Unit tests
├── configs/                # Configuration files
└── outputs/                # Model outputs, logs, checkpoints
```

## Setup

```bash
# Create virtual environment
python -m venv venv
source venv/bin/activate  # On macOS/Linux

# Install dependencies
pip install -r requirements.txt
```
